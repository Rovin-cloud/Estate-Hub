Build a full-stack Real Estate CRM SaaS MVP using Next.js (App Router).

You are a Senior Full Stack SaaS Developer.

PROJECT:
Real Estate CRM System for Bangladesh

GOAL:
Build a working CRM where real estate companies can manage leads, customers, properties, tasks, and sales pipeline.

---

IMPORTANT RULE (VERY STRICT):

Use ONLY these environment variables for Supabase:

NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY

Do NOT use any other env naming like VITE_ or SERVER keys.

---

ENV SETUP:

Create a .env.example file:

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

Ensure the app reads these variables correctly in Next.js.

Supabase client setup must be:

import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL,
  process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY
)
